# Grid

A simple grid based layout component.

```sh
npm install @vrembem/grid
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/grid";
  ```
</CodeExample>

## Config options

Grid comes with the following configuration options that can be set using either the [config module](/packages/core/config#using-config-module) or [Sass with clause](/packages/core/config#using-with-clause) methods.

<CodeExpander>
```scss
$config: (
  // Used to build grid column modifiers and col utilities
  // @type number
  "grid-cols": 6,

  // Used to build grid row modifiers and row utilities
  // @type number
  "grid-rows": 6,

  // A map of gap modifiers to output
  // @type map
  "grid-gap": (
    "none": 0,
    "xs": 0.5em,
    "sm": 1em,
    "lg": 3em,
    "xl": 4em
  )
);
```
</CodeExpander>

## Basic usage

The most basic implementation of the grid layout consists of the grid container and it's children (grid items). Apply the `grid` class to the desired grid container with an optional `grid_cols_[value]` modifier to create a grid layout.

<CodeExample>
  <Fragment slot="output">
    <div class="layout-demo grid grid_cols_3">
      <div class="box">1</div>
      <div class="box">2</div>
      <div class="box">3</div>
    </div>
  </Fragment>
  ```html
  <div class="grid grid_cols_3">
    <div>...</div>
    <div>...</div>
    <div>...</div>
  </div>
  ```
</CodeExample>

### grid_inline

Also available is the `grid_inline` modifier. This sets the `display` property to `inline-grid` instead of `grid`.

<CodeExample>
  <Fragment slot="output">
    <div class="layout-demo grid grid_inline grid_cols_3">
      <div class="box">1</div>
      <div class="box">2</div>
      <div class="box">3</div>
    </div>
  </Fragment>
  ```html
  <div class="grid grid_inline grid_cols_3">
    <div>...</div>
    <div>...</div>
    <div>...</div>
  </div>
  ```
</CodeExample>

## grid_cols_[value]

By default the `grid` class only defines a single column. To add additional columns to the grid layout, use the `grid_cols_[value]` modifier. The number of variants available is controlled by the `grid.$cols` variable (defaults to `6`).

<CodeExample>
  <Fragment slot="demo">

      <option value="grid">None</option>
      <option value="grid grid_cols_1">grid_cols_1</option>
      <option value="grid grid_cols_2">grid_cols_2</option>
      <option value="grid grid_cols_3">grid_cols_3</option>
      <option value="grid grid_cols_4">grid_cols_4</option>
      <option value="grid grid_cols_5">grid_cols_5</option>
      <option value="grid grid_cols_6">grid_cols_6</option>
    </select>
  </Fragment>
  <Fragment slot="output">
    <div id="demo-grid-cols" class="layout-demo grid grid_cols_3">
      <div class="box">1</div>
      <div class="box">2</div>
      <div class="box">3</div>
      <div class="box">4</div>
      <div class="box">5</div>

    </div>
  </Fragment>

- `grid_cols_6`
