# Button

Buttons are a simple component that allow users to take actions.

```sh
npm install @vrembem/button
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/button";
  ```
</CodeExample>

## Basic usage

The most basic implementation of the button component consists of the `button` class that can be applied to `<button>`, `<a role="button">` or `<input>` elements with a type attribute of "button", "submit" or "reset". [Remember to add `role="button"`](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/button_role) to any none semantic buttons as needed.

<CodeExample>
  <div slot="output" class="level">
    <button class="button">Button</button>
    <a class="button" role="button" href="#">Anchor</a>
    <input class="button" type="button" value="Input" />
    <input class="button" type="submit" value="Submit" />
    <input class="button" type="reset" value="Reset" />
  </div>
  ```html
  <button class="button">Button</button>
  <a class="button" role="button">Anchor</a>
  <input class="button" type="button" value="Input" />
  <input class="button" type="submit" value="Submit" />
  <input class="button" type="reset" value="Reset" />
  ```
</CodeExample>

Elements within a button are centered vertically and spaced accordingly using the value set in `$gap` (defaults to `0.5em`). When elements are added inside a button, the button's text should also be wrapped in a `<span>` so that it can be spaced properly.

<CodeExample>
  <div slot="output">
    <button class="button">
      <Icon name="anchor" classes="icon_size_sm" />
      <span>Button</span>
      <span class="arrow"></span>
    </button>
  </div>
  ```html
  <button class="button">
    <svg class="icon icon_size_sm">...</svg>
    <span>Button</span>
    <span class="arrow"></span>
  </button>
  ```
</CodeExample>

### Disabled

When disabled using the `disabled` attribute, a button will inherit styles to visually appear noninteractive.

<CodeExample>
  <div slot="output">
    <button class="button" disabled>Button</button>
  </div>
  ```html
  <button class="button" disabled>Button</button>
  ```
</CodeExample>

### Loading

Buttons can also be given a loading state by adding the `is-loading` state class. This is useful when a button has an action that has a delayed response in order to give users a visual indicator that their action is being processed.

<CodeExample>
  <div slot="output">
    <button class="button is-loading" disabled>Button</button>
  </div>
  ```html
  <button class="button is-loading" disabled>Button</button>
  ```
</CodeExample>

> Warning: the `is-loading` state class will not work on `<input type="button">` buttons since it uses the `::after` pseudo-element to display the loading spinner.

## button_block_[value]

Gives a button "block" styles so that it spans the full width of its container. Values and class keys are generated using the [`$breakpoints`](#scss-variables-file) map. Using `button_block` will apply block styles in all viewports.

<CodeExample>
  <div slot="output">
    <button class="button button_block">Button</button>
  </div>
  ```html
  <button class="button button_block">Button</button>
  ```
</CodeExample>

**Available variants**

- `button_block`
- `button_block_xs`
- `button_block_sm`
- `button_block_md`
- `button_block_lg`
- `button_block_xl`

## button_color_[value]

Adds styles for changing the look and feel of a button. These are usually done with different background and text colors.

<CodeExample>
  <div slot="output" class="level">
    <button class="button button_color_primary">Button</button>
    <button class="button button_color_secondary">Button</button>
    <button class="button button_color_neutral">Button</button>
    <button class="button button_color_important">Button</button>
  </div>
  ```html
  <button class="button button_color_primary">Button</button>
  <button class="button button_color_secondary">Button</button>
  <button class="button button_color_neutral">Button</button>
  <button class="button button_color_important">Button</button>
  ```
</CodeExample>

**Available variants**

- `button_color_primary`
- `button_color_secondary`
- `button_color_neutral`
- `button_color_important`

## button_icon

Adds styles that make icon-only buttons have equal padding. Will be square if the icon used also has equal width and height.

<CodeExample>
  <div slot="output">
    <button class="button button_icon">
      <Icon name="anchor" />
    </button>
  </div>
  ```html
  <button class="button button_icon">
    <svg class="icon">...</svg>
  </button>
  ```
</CodeExample>

## button_size_[value]

Adjust the size of a button by increasing or decreasing its padding and font-size.

<CodeExample>
  <div slot="output" class="level">
    <button class="button button_size_sm">Button</button>
    <button class="button">Button</button>
    <button class="button button_size_lg">Button</button>
  </div>
  ```html
  <button class="button button_size_sm">Button</button>
  <button class="button button_size_lg">Button</button>
  ```
</CodeExample>

**Available variants**

- `button_size_sm`
- `button_size_lg`

## Customization

Buttons are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="button/src/_root.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";
  @use "./variables" as var;

  :root {
    @include core.css("button-size", var.$size);
    @include core.css("button-padding", var.$padding);
    @include core.css("button-gap", var.$gap);
    @include core.css("button-background", var.$background);
    @include core.css("button-background-hover", var.$background-hover);
    @include core.css("button-background-active", var.$background-active);
    @include core.css("button-foreground", var.$foreground);
    @include core.css("button-foreground-hover", var.$foreground-hover);
    @include core.css("button-foreground-active", var.$foreground-active);
    @include core.css("button-border-color", var.$border-color);
    @include core.css("button-border-color-hover", var.$border-color-hover);
    @include core.css("button-border-color-active", var.$border-color-active);
    @include core.css("button-border-width", var.$border-width);
    @include core.css("button-border-radius", var.$border-radius);
    @include core.css("button-box-shadow", var.$box-shadow);
    @include core.css("button-box-shadow-focus", var.$box-shadow-focus);
    @include core.css("button-font-size", var.$font-size);
    @include core.css("button-font-weight", var.$font-weight);
    @include core.css("button-line-height", var.$line-height);
    @include core.css("button-transition-property", var.$transition-property);
    @include core.css("button-transition-duration", var.$transition-duration);
    @include core.css("button-transition-timing-function", var.$transition-timing-function);
    @include core.css("button-disabled-opacity", var.$disabled-opacity);
    @include core.css("button-loading-size", var.$loading-size);
    @include core.css("button-loading-border", var.$loading-border);
    @include core.css("button-loading-animation-duration", var.$loading-animation-duration);
    @include core.css("button-loading-animation-timing-function", var.$loading-animation-timing-function);
  }
  ```
</DocBlock>

  ```scss
  @use "sass:list";
  @use "@vrembem/core";

  $background-hover: transparent !default;
  $background-active: transparent !default;
  $foreground: theme.get("foreground") !default;
  $foreground-hover: theme.get("foreground") !default;
  $foreground-active: theme.get("foreground") !default;
  $border-color: theme.get("border-color-dark") !default;
  $border-color-hover: theme.get("border-color-darker") !default;
  $border-color-active: theme.get("border-color-darker") !default;
  $border-width: 1px !default;
  $border-radius: core.$border-radius !default;
  $box-shadow: 0 0 0 0 rgb(0 0 0 / 0%) !default;
  $box-shadow-focus: 0 0 0 0.2rem rgb(0 0 0 / 15%) !default;
  $font-size: 1em !default;
  $font-weight: inherit !default;
  $line-height: 1.6 !default;

  // Transition
  $transition-property: background, color, border-color, box-shadow !default;

  // Disabled
  $disabled-opacity: 0.6 !default;

  // is-loading
  $loading-size: 1em !default;
  $loading-border: 2px solid !default;
  $loading-border-tpl: 1 1 0 0;
  $loading-animation-duration: 0.6s !default;
  $loading-animation-timing-function: linear !default;

  $icon-padding-sm: calc(#{list.nth(core.$form-control-padding-sm, 1)} - 1px);

  // button_size_sm
  $size-sm: core.$form-control-size-sm !default;
  $size-sm-padding: calc(#{list.nth(core.$form-control-padding-sm, 1)} - 1px) list.nth(core.$form-control-padding-sm, 2) !default;
  $size-sm-font-size: core.$font-size-sm !default;
  $size-sm-line-height: core.css("line-height-sm") !default;

  ```
