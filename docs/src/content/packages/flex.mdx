# Flex

A simple flexbox based layout component.

```sh
npm install @vrembem/flex
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/flex";
  ```
</CodeExample>

## Config options

Flex comes with the following configuration options that can be set using either the [config module](/packages/core/config#using-config-module) or [Sass with clause](/packages/core/config#using-with-clause) methods.

<CodeExpander>
```scss
$config: (
  // A map of gap modifiers to output
  // @type map
  "flex-gap": (
    "none": 0,
    "xs": 1px,
    "sm": 0.5em,
    "lg": 1.5em,
    "xl": 2em
  ),

  // The range of flex-grow classes to output
  // @type number
  "flex-grow-count": 6,

  // The range of flex-shrink classes to output
  // @type number
  "flex-shrink-count": 6
);
```
</CodeExpander>

## Customization

Flex is primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="flex/src/_root.scss">

```scss
:root {
  --vb-flex-gap: 1em;
  --vb-flex-wrap: wrap;
}
```
</DocBlock>

<DocBlock name="SCSS Variables" src="flex/src/_variables.scss">

```scss
@use "@vrembem/core";
@use "@vrembem/core/css";

$breakpoints: core.$breakpoints !default;
$gap: 1em !default;
$gap-map: (
  "none": 0,
  "xs": 1px,
  "sm": .5em,
  "md": 1em,
  "lg": 1.5em,
  "xl": 2em,
) !default;
$wrap: wrap !default;
$order-count: 6 !default;
$grow-count: 6 !default;
$shrink-count: 6 !default;

@include css.set("flex", (
  "gap": $gap,
  "gap-x": null,
  "gap-y": null,
  "wrap": $wrap,
```
