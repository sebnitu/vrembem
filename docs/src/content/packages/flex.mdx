# Flex

A simple flexbox based layout component.

```sh
npm install @vrembem/flex
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/flex";
  ```
</CodeExample>

## Basic usage

The most basic implementation of the flex utility component consists of a container element with the `flex` class. This can contain any number of children elements.

<CodeExample tabs>
  <Fragment slot="output">
    <div class="flex">
      <div class="box">One</div>
      <div class="box">Two</div>
      <div class="box">Three</div>
      <div class="box">Four</div>
      <div class="box">Five</div>
      <div class="box">Six</div>
      <div class="box">Seven</div>
      <div class="box">Eight</div>
      <div class="box">Nine</div>
      <div class="box">Ten</div>
    </div>
  </Fragment>
  <Fragment slot="tab-html">
  ```html
  <div class="flex">
    ...
  </div>
  ```
  </Fragment>
  <Fragment slot="tab-css">
  ```css
  .flex {
    display: flex;
    gap: var(--vb-flex-gap-y, var(--vb-flex-gap)) var(--vb-flex-gap-x, var(--vb-flex-gap));
    flex-wrap: var(--vb-flex-wrap);
  }
  ```
  </Fragment>
</CodeExample>

### flex_inline

Also available is the `flex_inline` modifier. This sets the `display` property to `inline-flex` instead of `flex`.

<CodeExample>
  <Fragment slot="output">
    <div class="flex flex_inline">
      <div class="box">One</div>
      <div class="box">Two</div>
      <div class="box">Three</div>
      <div class="box">Four</div>
      <div class="box">Five</div>
    </div>
  </Fragment>
  ```html
  <div class="flex flex_inline">
    ...
  </div>
  ```
</CodeExample>

## flex-gap-[key]

      <option value="flex-gap-md">flex-gap-md</option>
      <option value="flex-gap-lg">flex-gap-lg</option>
      <option value="flex-gap-xl">flex-gap-xl</option>
    </select>
  </Fragment>

      <div class="box">Three</div>
      <div class="box">Four</div>
      <div class="box">Five</div>
      <div class="box">Six</div>
      <div class="box">Seven</div>


**Available variants**

## flex-wrap

This utility toggles flexbox wrapping so that flex items are forced onto one line or can wrap onto multiple lines. By default, `flex` has wrapping enabled but this can be changed via the `--vb-flex-wrap` custom property or the `flex.$wrap` SCSS variable.

      <option value="flex-wrap-reverse">flex-wrap-reverse</option>
    </select>
  </Fragment>
  <Fragment slot="output">

      <div class="box">Ten</div>

## flex-items-[value]

This utility helps to control the flex property on flex items (direct children of flex containers). The variants available allow for the most common combinations of flex grow, shrink and basis properties.

<CodeExample>
  <Fragment slot="demo">
    <label class="text-nowrap" for="demo-flex-items-select">Variants Demo:</label>
    <select id="demo-flex-items-select" class="input input_type_select input_size_sm input_auto" data-demo-target="demo-flex-items">
      <option value="flex-items-initial">flex-items-initial</option>
      <option value="flex-items-auto">flex-items-auto</option>
      <option value="flex-items-none">flex-items-none</option>
      <option value="flex-items-equal">flex-items-equal</option>
      <option value="flex-items-fill">flex-items-fill</option>
      <option value="flex-items-full">flex-items-full</option>
    </select>
  </Fragment>
  <Fragment slot="output">
    <div id="demo-flex-items" class="flex flex-items-auto">
      <div class="box">1. One</div>
      <div class="box">2. Two</div>
      <div class="box">3. Three</div>
    </div>
  </Fragment>
  ```html
  <div class="flex flex-items-auto">
    <div>...</div>
    <div>...</div>
    <div>...</div>
  </div>
  ```
</CodeExample>

**Available variants**

- `flex-items-initial`: Sets flex items to `flex: initial;`.
- `flex-items-auto`: Sets flex items to `flex: auto;`.
- `flex-items-none`: Sets flex items to `flex: equal;`.
- `flex-items-equal`: Sets flex items to `flex: 1 1 0;`.
- `flex-items-fill`: Sets flex items to `flex: 1 0 auto;`.
- `flex-items-full`: Sets flex items to `flex: 1 0 100%;`.

## Customization

Flex is primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="flex/src/_root.scss">

```scss
:root {
  --vb-flex-gap: 1em;
  --vb-flex-wrap: wrap;
}
```
</DocBlock>

<DocBlock name="SCSS Variables" src="flex/src/_variables.scss">

```scss
@use "@vrembem/core";
@use "@vrembem/core/css";

$breakpoints: core.$breakpoints !default;
$gap: 1em !default;
$gap-map: (
  "none": 0,
  "xs": 1px,
  "sm": .5em,
  "md": 1em,
  "lg": 1.5em,
  "xl": 2em,
) !default;
$wrap: wrap !default;
$order-count: 6 !default;
$grow-count: 6 !default;
$shrink-count: 6 !default;

@include css.set("flex", (
  "gap": $gap,
  "gap-x": null,
  "gap-y": null,
  "wrap": $wrap,
));
```
</DocBlock>
