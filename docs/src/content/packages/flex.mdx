# Flex

A simple flexbox utility component.

```sh
npm install @vrembem/flex
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/flex";
  ```
</CodeExample>

## Basic usage

The most basic implementation of the flex utility component consists of a container element with the `flex` class. This can contain any number of children elements.

<CodeExample tabs>
  <Fragment slot="output">
    <div class="flex">
      <div class="box">One</div>
      <div class="box">Two</div>
      <div class="box">Three</div>
      <div class="box">Four</div>
      <div class="box">Five</div>
      <div class="box">Six</div>
      <div class="box">Seven</div>
      <div class="box">Eight</div>
      <div class="box">Nine</div>
      <div class="box">Ten</div>
    </div>
  </Fragment>
  <Fragment slot="tab-html">
  ```html
  <div class="flex">
    <div>...</div>
    <div>...</div>
    <div>...</div>
  </div>
  ```
  </Fragment>
  <Fragment slot="tab-css">
  ```css
  .flex {
    display: flex;
    gap: var(--vb-flex-gap-y, var(--vb-flex-gap)) var(--vb-flex-gap-x, var(--vb-flex-gap));
    flex-wrap: var(--vb-flex-wrap);
  }
  ```
  </Fragment>
</CodeExample>

### flex-inline

Also available is the `flex-inline` utility. This applies the same styles as `flex` but will set the `display` property to `inline-flex` instead of `flex`.

<CodeExample tabs>

      <div class="box">Two</div>

      <div class="box">Five</div>
    </div>
  </Fragment>
  <Fragment slot="tab-html">

  ```html
  <div class="flex-inline">
    <div>...</div>
    <div>...</div>
    <div>...</div>
  ```

</CodeExample>

## flex-gap-[key]

## flex-wrap

This utility toggles flexbox wrapping so that flex items are forced onto one line or can wrap onto multiple lines. By default, `flex` and `flex-inline` has wrapping enabled but this can be changed via the `--vb-flex-wrap` custom property or the `flex.$wrap` SCSS variable.

    <select id="demo-flex-wrap-select" class="input input_auto input_size_sm input_type_select" data-demo-target="demo-flex-wrap">
      <option value="flex-nowrap">flex-nowrap</option>
      <option value="flex-wrap">flex-wrap</option>
      <option value="flex-wrap-reverse">flex-wrap-reverse</option>
    </select>
  </Fragment>
  <Fragment slot="output">
    <div id="demo-flex-wrap" class="flex flex-nowrap">
      <div class="box">One</div>
      <div class="box">Two</div>

      <div class="box">Five</div>
      <div class="box">Six</div>
      <div class="box">Seven</div>
      <div class="box">Eight</div>
      <div class="box">Nine</div>
      <div class="box">Ten</div>


**Available variants**

## Customization

Level is primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="flex/src/_root.scss">

```scss
:root {
  --vb-flex-gap: 1em;
  --vb-flex-wrap: wrap;
}
```
</DocBlock>

<DocBlock name="SCSS Variables" src="flex/src/_variables.scss">

```scss
@use "@vrembem/core";
@use "@vrembem/core/css";

$breakpoints: core.$breakpoints !default;
$gap: 1em !default;
$gap-map: (
  "none": 0,
  "xs": 1px,
  "sm": .5em,
  "md": 1em,
  "lg": 1.5em,
  "xl": 2em,
) !default;
$wrap: wrap !default;
$order-count: 12 !default;
$grow-count: 6 !default;
$shrink-count: 6 !default;

@include css.set("flex", (
  "gap": $gap,
  "gap-x": null,
  "gap-y": null,
  "wrap": $wrap,
));
```
</DocBlock>
