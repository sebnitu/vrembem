# Table

A table component for displaying HTML tables.

```sh
npm install @vrembem/table
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/table";
  ```
</CodeExample>

## Basic usage

For basic table styles, only the `table` component class is required. Use [proper HTML table markup](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table) and styles should apply as expected.

## table_responsive_[key]

When the [`scroll-box` base module](/packages/base#scroll-box) isn't mobile friendly enough, this modifier adds responsive styles to a table so that it's easier to read on specific breakpoints. This is done by using the `data-mobile-label` attribute on table cells that should correspond to the table column header for that specific cell. Use the value-less `table_responsive` modifier to apply these styles universally.

<CodeExample>
  <Fragment slot="output">
    <table class="table table_responsive table_style_bordered">
      <thead>
        <tr>
          <th>Name</th>
          <th>Mass</th>
          <th>Class</th>
          <th>Variant</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td data-mobile-label="Name">Jenner</td>
          <td data-mobile-label="Mass">35t</td>
          <td data-mobile-label="Class">Light</td>
          <td data-mobile-label="Variant">JR7-F</td>
        </tr>
        <tr>
          <td data-mobile-label="Name">Hunchback</td>
          <td data-mobile-label="Mass">50t</td>
          <td data-mobile-label="Class">Medium</td>
          <td data-mobile-label="Variant">HBK-4P</td>
        </tr>
      </tbody>
    </table>
  </Fragment>
  ```html
  <table class="table table_responsive">
    <thead>
      <tr>
        <th>key</th>
        <th>value</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td data-mobile-label="key">...</td>
        <td data-mobile-label="value">...</td>
      </tr>
    </tbody>
  </table>
  ```
</CodeExample>

The mobile label value will be truncated if it exceeds the width set by `$mobile-label-width`, so keep the label short when possible.

**Available variants**

- `table_responsive`
- `table_responsive_xs`
- `table_responsive_sm`
- `table_responsive_md`
- `table_responsive_lg`
- `table_responsive_xl`

### `@mixin table-responsive-styles()`

Output responsive table styles. This is used to generate the styles for the `table_responsive_[key]` modifier.

**Example**

```scss
.table_custom_modifier {
  @include table-responsive-styles();
}
```

## Customization

Tables are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="table/src/_root.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";
  @use "./variables" as var;

  #{theme.$root-selector} {
    @include core.css("table-padding", var.$padding);
    @include core.css("table-background", var.$background);
    @include core.css("table-background-zebra", var.$background-zebra);
    @include core.css("table-background-hover", var.$background-hover);
    @include core.css("table-foreground", var.$foreground);
    @include core.css("table-foreground-hover", var.$foreground-hover);
    @include core.css("table-border", var.$border);
    @include core.css("table-border-alt", var.$border-alt);
    @include core.css("table-mobile-label-width", var.$mobile-label-width);
    @include core.css("table-mobile-label-spacing", var.$mobile-label-spacing);
    @include core.css("table-mobile-label-background", var.$mobile-label-background);
    @include core.css("table-mobile-label-foreground", var.$mobile-label-foreground);
  }

  @include theme.output("table");
  ```
</DocBlock>

<DocBlock name="SCSS Variables" src="table/src/_variables.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/palette";
  @use "@vrembem/core/theme";

  $breakpoints: core.$breakpoints !default;
  $padding: 0.5em 0.75em !default;
  $background: null !default;
  $background-zebra: null !default;
  $background-hover: null !default;
  $foreground: null !default;
  $foreground-hover: null !default;
  $border: null !default;
  $border-alt: null !default;

  // table_responsive
  $mobile-label-attr: "data-mobile-label" !default;
  $mobile-label-width: 8em !default;
  $mobile-label-spacing: core.get-side($padding, left) !default;
  $mobile-label-background: null !default;
  $mobile-label-foreground: null !default;

  // table_size_sm
  $size-sm-padding: 0.25em 0.5em !default;
  $size-sm-mobile-label-width: $mobile-label-width !default;
  $size-sm-mobile-label-spacing: core.get-side($size-sm-padding, left) !default;

  // table_size_lg
  $size-lg-padding: 1em 1.25em !default;
  $size-lg-mobile-label-width: $mobile-label-width !default;
  $size-lg-mobile-label-spacing: core.get-side($size-lg-padding, left) !default;

  // Theme maps
  $theme-light: (
    "background": theme.get("background"),
    "background-zebra": palette.get("neutral", 50%, 6%),
    "background-hover": palette.get("primary", 50%, 20%),
    "border": 1px solid theme.get("border-color"),
    "border-alt": 1px dotted theme.get("border-color"),
    "foreground": theme.get("foreground"),
    "foreground-hover": null
  ) !default;

  $theme-dark: (
    "background": theme.get("background"),
    "background-zebra": palette.get("neutral", 50%, 10%),
    "background-hover": palette.get("primary", 50%, 20%),
    "border": 1px solid theme.get("border-color-dark"),
    "border-alt": 1px dotted theme.get("border-color-dark"),
    "foreground": theme.get("foreground"),
    "foreground-hover": null
  ) !default;

  @include theme.set("light", "table", $theme-light);
  @include theme.set("dark", "table", $theme-dark);
  ```
