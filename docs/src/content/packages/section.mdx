# Section

A container component for wrapping distinct sections of a page.

```sh
npm install @vrembem/section
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/section";
  ```
</CodeExample>

## Basic usage

Sections are composed using the base wrapper (`section`) along with three optional elements. The `section__container` element's primary purpose is to wrap content with a max-width and center using margin auto. The other two element's—`section__image` and `section__screen`—add a background image and colored screen that fills the full size of the section component.

- `section`
  - `section__container`
  - `section__image`
  - `section__screen`

<CodeExample flush={true}>
  <Fragment slot="output">
    <section class="section">
      <div class="section__container gap-y text-align-center">
        <h1 class="h1">Section example</h1>
        <p>This is a demonstration of the section component using a screen and background image. The screen, background and foreground colors can be adjusted using CSS variables.</p>
        <hr class="sep margin-y-xl" />
        <div class="level flex-justify-center">
          <button class="button button_color_primary">First action</button>
          <button class="button button_color_secondary">Second action</button>
        </div>
      </div>
      <img class="section__image" src="https://picsum.photos/id/335/600/300" />
      <div class="section__screen"></div>
    </section>
  </Fragment>
  ```html
  <section class="section">
    <div class="section__container">
      ...
    </div>
    <img class="section__image" src="..." />
    <div class="section__screen"></div>
  </section>
  ```
</CodeExample>

Section screens and backgrounds are displayed behind the section container. These can be paired with the `vb-theme-light` or `vb-theme-dark` theme classes to create a light or dark section theme.

## section_size_[key]

Sections have a few size modifier options to help adjust the space that is used based on how prominent the section needs to be. These are optimized for all screen sizes to avoid oversized areas on mobile.

<CodeExample flush={true}>
  <Fragment slot="output">
    <section class="section section_size_xl">
      <div class="section__container gap-y text-align-center">
        <h1 class="h1">Section example</h1>
        <p>This is a demonstration of the section component using a screen and background image. The screen, background and foreground colors can be adjusted using CSS variables.</p>
        <hr class="sep margin-y-xl" />
        <div class="level flex-justify-center">
          <button class="button button_color_primary">First action</button>
          <button class="button button_color_secondary">Second action</button>
        </div>
      </div>
      <img class="section__image" src="https://picsum.photos/id/335/600/300" />
      <div class="section__screen"></div>
    </section>
  </Fragment>
  ```html
  <section class="section section_size_xl">
    ...
  </section>
  ```
</CodeExample>

**Available Variants**

* `section_size_sm`
* `section_size_md`
* `section_size_lg`
* `section_size_xl`

## Customization

Sections are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" type="file" src="section/src/_root.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";
  @use "./variables" as var;

  #{theme.$root_selector} {
    @include core.css("section-max-width", var.$max-width);
    @include core.css("section-background", var.$background);
    @include core.css("section-foreground", var.$foreground);
    @include core.css("section-image-opacity", var.$image-opacity);
    @include core.css("section-screen-background", var.$screen-background);
    @include core.css("section-screen-opacity", var.$screen-opacity);
    @include core.css-query("section-padding", var.$padding, var.$breakpoints);
  }

  @include theme.output("section");
  ```
</DocBlock>

<DocBlock name="SCSS Variables" type="file" src="section/src/_variables.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";

  $breakpoints: core.$breakpoints !default;
  $max-width: 70rem !default;
  $padding: (
    "sm": 1.5em,
    "md": 2em 1.5em,
    "lg": 3em 1.5em,
  ) !default;
  $background: null !default;
  $foreground: null !default;
  $image-opacity: 1 !default;
  $screen-background: null !default;
  $screen-opacity: null !default;

  // section_size_[key]
  $size-xs-padding: null !default;
  $size-sm-padding: 1em !default;
  $size-md-padding: (
    "sm": 1.5em,
    "md": 2em 1.5em,
    "lg": 3em 1.5em
  ) !default;
  $size-lg-padding: (
    "sm": 1.5em,
    "md": 4em 1.5em,
    "lg": 6em 1.5em
  ) !default;
  $size-xl-padding: (
    "xs": 1.5em,
    "sm": 3em 1.5em,
    "md": 8em 2em,
    "lg": 12em 2em
  ) !default;

  // Theme maps
  $theme-light: (
    "background": theme.get("background"),
    "foreground": theme.get("foreground"),
    "screen-background": theme.get("background"),
    "screen-opacity": 0.8
  ) !default;

  $theme-dark: (
    "background": theme.get("background"),
    "foreground": theme.get("foreground"),
    "screen-background": theme.get("background"),
    "screen-opacity": 0.7
  ) !default;

  @include theme.set("light", "section", $theme-light);
  @include theme.set("dark", "section", $theme-dark);
  ```
</DocBlock>
