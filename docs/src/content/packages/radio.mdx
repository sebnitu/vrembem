# Radio

Radios allow the user to select a single option from a set.

```sh
npm install @vrembem/radio
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/radio";
  ```
</CodeExample>

## Basic usage

Radio buttons are composed using a set of `<span>` elements alongside the native `<input type="radio">` element which should be given the `radio__native` class and come before the remaining presentational `<span>` elements.

<CodeExample>
  <Fragment slot="output">
    <span class="radio">
      <input type="radio" class="radio__native" name="example-1" value="1" checked />
      <span class="radio__background">
        <span class="radio__circle">
          <span class="radio__dot"></span>
        </span>
      </span>
    </span>
    <span class="radio">
      <input type="radio" class="radio__native" name="example-1" value="2" />
      <span class="radio__background">
        <span class="radio__circle">
          <span class="radio__dot"></span>
        </span>
      </span>
    </span>
  </Fragment>
  ```html
  <span class="radio">
    <input type="radio" class="radio__native">
    <span class="radio__background">
      <span class="radio__circle">
        <span class="radio__dot"></span>
      </span>
    </span>
  </span>
  ```
</CodeExample>

## radio_size_[value]

## Customization

Radios are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="radio/src/_root.scss">

  ```scss
  :root {
    --vb-radio-size: var(--vb-form-control-size);
    --vb-radio-circle-size: 20px;
    --vb-radio-dot-size: 8px;
    --vb-radio-color: var(--vb-primary-50);
    --vb-radio-border-width: 2px;
    --vb-radio-background-border-radius: var(--vb-border-radius-circle);
    --vb-radio-background-opacity: 0%;
    --vb-radio-background-opacity-hover: 20%;
    --vb-radio-background-opacity-focus: 20%;
    --vb-radio-background-opacity-active: 30%;
    --vb-radio-transition-duration: var(--vb-transition-duration-short);
    --vb-radio-transition-timing-function: var(--vb-transition-timing-function);
  }

  :root, .vb-theme-root, .vb-theme-light {
    --vb-radio-fill: white;
    --vb-radio-border-color: var(--vb-neutral-50);
    --vb-radio-dot-color: white;
  }

  @media (prefers-color-scheme: dark) {
    :root, .vb-theme-root {
      --vb-radio-fill: var(--vb-neutral-10);
      --vb-radio-border-color: var(--vb-neutral-80);
      --vb-radio-dot-color: var(--vb-neutral-10);
    }
  }

  .vb-theme-dark {
    --vb-radio-fill: var(--vb-neutral-10);
    --vb-radio-border-color: var(--vb-neutral-80);
    --vb-radio-dot-color: var(--vb-neutral-10);
  }
  ```
</DocBlock>

<DocBlock name="SCSS Variables" src="radio/src/_variables.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/css";
  @use "@vrembem/core/palette";
  @use "@vrembem/core/theme";

  $size: css.get("form-control", "size") !default;
  $circle-size: 20px !default;
  $dot-size: 8px !default;
  $dot-color: null !default;
  $color: palette.get("primary") !default;
  $fill: null !default;
  $border-color: null !default;
  $border-width: 2px !default;
  $background-border-radius: css.get("border-radius-circle") !default;
  $background-opacity: 0% !default;
  $background-opacity-hover: 20% !default;
  $background-opacity-focus: 20% !default;
  $background-opacity-active: 30% !default;

  // Transition
  $transition-duration: css.get("transition", "duration-short") !default;
  $transition-timing-function: css.get("transition", "timing-function") !default;

  // radio_size_sm
  $size-sm: css.get("form-control", "size-sm") !default;
  $size-sm-border-width: 2px !default;
  $size-sm-circle: 16px !default;
  $size-sm-dot: 6px !default;

  // radio_size_lg
  $size-lg: css.get("form-control", "size-lg") !default;
  $size-lg-border-width: 2.5px !default;
  $size-lg-circle: 26px !default;
  $size-lg-dot: 10px !default;

  // Custom property
  @include css.set("radio", (
    "size": $size,
    "circle-size": $circle-size,
    "dot-size": $dot-size,
    "dot-color": $dot-color,
    "color": $color,
    "fill": $fill,
    "border-color": $border-color,
    "border-width": $border-width,
    "background": (
      "border-radius": $background-border-radius,
      "opacity": $background-opacity,
      "opacity-hover": $background-opacity-hover,
      "opacity-focus": $background-opacity-focus,
      "opacity-active": $background-opacity-active,
    ),
    "transition": (
      "duration": $transition-duration,
      "timing-function": $transition-timing-function,
    )
  ));

  // Theme maps
  $theme-light: (
    "fill": white,
    "border-color": palette.get("neutral"),
    "dot-color": white
  ) !default;

  $theme-dark: (
    "fill": palette.get("neutral", 10),
    "border-color": palette.get("neutral", 80),
    "dot-color": palette.get("neutral", 10)
  ) !default;

  @include theme.set("light", "radio", $theme-light);
  @include theme.set("dark", "radio", $theme-dark);
  ```
</DocBlock>
