The `core` package provides a number of pre-defined CSS variables that are used throughout the Vrembem library. These variables are not output when you import or use `@vrembem/core`. To output core CSS varibales, use the `@vrembem/core/root` export.

<CodeExample lang="scss">
```scss
@forward "@vrembem/core/root";
```
</CodeExample>

Alternatively, custom properties can also be output using the `core/css` and `core/theme` modules:

<CodeExample lang="scss">
```scss
@use "@vrembem/core/css";
@use "@vrembem/core/theme";

// Output all registered CSS variables
:root {
  @include css.output;
}

// Output all registered theme CSS variables
@include theme.output;
```
</CodeExample>
