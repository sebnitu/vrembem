The following is a reference for the `CollectionEntry` class API, which is the basis for all Vrembem JavaScript components.

## Constructor

The `CollectionEntry` constructor is run whenever a new entry is added to a collection.

<CodeExample lang="ts">
```ts
new CollectionEntry(options?: Record<string, any>);
```
</CodeExample>
