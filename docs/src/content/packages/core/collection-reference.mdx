The following is a reference for the `Collection` class API, which is the basis for all Vrembem JavaScript components.

## Constructor

The `Collection` constructor is run when a new collection is instantiated.

<CodeExample lang="ts">
```ts
new Collection(options?: Record<string, any>);
```
</CodeExample>
