# Notice

A component for highlighting messages to the user.

```sh
npm install @vrembem/notice
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/notice";
  ```
</CodeExample>

## Basic usage

The most basic implementation component consists of the `notice` container element wrapping text content.

<CodeExample>
  <Fragment slot="output">
    <div class="notice">
      <p>This is an example notice.</p>
    </div>
  </Fragment>
  ```html
  <div class="notice">
    <p>...</p>
  </div>
  ```
</CodeExample>

Add a title to your notice using the `notice__title` element.

<CodeExample>
  <Fragment slot="output">
    <div class="notice">
      <h2 class="notice__title">Notice title</h2>
      <p>This is an example notice with a title. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus libero est, fermentum ac risus et.</p>
    </div>
  </Fragment>
  ```html
  <div class="notice">
    <h2 class="notice__title">...</h2>
    <p>...</p>
  </div>
  ```
</CodeExample>

### Dismissible

It's a common pattern to allow users to dismiss notices. This can be accomplished using the button component along with some simple vanilla JavaScript.

<CodeExample tabs resettable>
  <Fragment slot="output">
    <div class="notice">
      <div class="flex flex-justify-between flex-align-center">
        <p>This is an example notice with a dismiss button.</p>
        <button class="button button_icon button_size_sm" aria-label="Dismiss this notice" data-dismiss=".notice">
          <Icon name="x" classes="icon_size_sm" />
        </button>
      </div>
    </div>
  </Fragment>
  <Fragment slot="tab-html">
    ```html
    <div class="notice">
      <div class="flex flex-justify-between">
        <p>...</p>
        <button 
          class="button button_icon button_size_sm" 
          aria-label="Dismiss this notice" 
          data-dismiss=".notice"
        >
          ...
        </button>
      </div>
    </div>
    ```
  </Fragment>
  <Fragment slot="tab-js">
    ```js
    // Get the button elements.
    const btns = document.querySelectorAll("[data-dismiss]");

    // Loop through them and add click event listeners.
    btns.forEach((btn) => {
      btn.addEventListener("click", (event) => {
        // Get the value of data-dismiss and select the nearest matching element.
        const selector = btn.getAttribute("data-dismiss");
        const result = event.target.closest(selector);

        // Hide the notice if it exists.
        if (result) result.classList.add("display-none");
      });
    });
    ```
  </Fragment>
</CodeExample>

## notice_color_[value]

Adds styles for changing the look and feel to better reflect the urgency or status of a notice.

<CodeExample>

      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus libero est, fermentum ac risus et.</p>

      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus libero est, fermentum ac risus et.</p>

**Available variants**

- `notice_color_secondary`
- `notice_color_neutral`
- `notice_color_important`

## Customization

Notices are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="notice/src/_root.scss">

```scss
:root {
  --vb-notice-gap: 0.5em;
  --vb-notice-padding: 1em;
  --vb-notice-border-radius: var(--vb-border-radius);
  --vb-notice-title-font-size: var(--vb-font-size-lg);
  --vb-notice-title-line-height: var(--vb-line-height-lg);
  --vb-notice-title-font-weight: var(--vb-font-weight-semi-bold);
}

:root, .vb-theme-root, .vb-theme-light {
  --vb-notice-background: var(--vb-neutral-95);
  --vb-notice-foreground: var(--vb-neutral-20);
}

@media (prefers-color-scheme: dark) {
  :root, .vb-theme-root {
    --vb-notice-background: var(--vb-neutral-20);
    --vb-notice-foreground: var(--vb-neutral-95);
  }
}

.vb-theme-dark {
  --vb-notice-background: var(--vb-neutral-20);
  --vb-notice-foreground: var(--vb-neutral-95);
}
```
</DocBlock>

<DocBlock name="SCSS Variables" src="notice/src/_variables.scss">

```scss
@use "@vrembem/core";
@use "@vrembem/core/css";
@use "@vrembem/core/theme";
@use "@vrembem/core/palette";

$gap: 0.5em !default;
$padding: 1em !default;
$background: null !default;
$foreground: null !default;
$border: null !default;
$border-radius: css.get("border-radius") !default;
$box-shadow: null !default;
$title-font-size: css.get("font-size-lg") !default;
$title-line-height: css.get("line-height-lg") !default;
$title-font-weight: css.get("font-weight-semi-bold") !default;

@include css.set("notice", (
  "gap": $gap,
  "padding": $padding,
  "background": $background,
  "foreground": $foreground,
  "border": $border,
  "border-radius": $border-radius,
  "box-shadow": $box-shadow,
  "title": (
    "font-size": $title-font-size,
    "line-height": $title-line-height,
    "font-weight": $title-font-weight,
  ),
));

$theme-light: (
  "background": palette.get("neutral", 95),
  "foreground": palette.get("neutral", 20),
  "border": null,
  "box-shadow": null
) !default;

$theme-dark: (
  "background": palette.get("neutral", 20),
  "foreground": palette.get("neutral", 95),
  "border": null,
  "box-shadow": null
) !default;

@include theme.set("light", "notice", $theme-light);
@include theme.set("dark", "notice", $theme-dark);
```
</DocBlock>
