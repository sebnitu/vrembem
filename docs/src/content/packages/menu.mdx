# Menu

Menus represent groups of links, actions or tools that a user can interact with.

```sh
npm install @vrembem/menu
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/menu";
  ```
</CodeExample>

## Basic usage

The menu component is composed of at minimum three parts: `menu`, `menu__item` and `menu__action`. The menu and menu items should be a `<ul>` and list items `<li>` respectively while the menu action can be either an `<a>` or `<button>` element. Also available is the optional `menu__sep` element to create separators in between menu items.

<CodeExample>
  <Fragment slot="output">
    <Menu name="edit" />
  </Fragment>
  ```html
  <ul class="menu">
    <li class="menu__item">
      <button class="menu__action">
        ...
      </button>
    </li>
    <li class="menu__sep"></li>
  </ul>
  ```
</CodeExample>

Use the `flex-grow` utility class to wrap text inside the `menu__action` element. Additional elements inside the menu action receive appropriate spacing.

<CodeExample>
  <Fragment slot="output">
    <Menu name="edit-all" />
  </Fragment>
  ```html
  <ul class="menu">
    <li class="menu__item">
      <button class="menu__action">
        <span>...</span>
        <span class="flex-grow">...</span>
        <span>...</span>
      </button>
    </li>
    ...
  </ul>
  ```
</CodeExample>

For links that only contain an icon, you can use the `menu__action_icon` element modifier to create a square link similar to the `button_icon` modifier.

<CodeExample>
  <Fragment slot="output">
    <Menu name="browser" />
  </Fragment>
  ```html
  <ul class="menu menu_inline">
    <li class="menu__item">
      <button class="menu__action menu__action_icon">
        ...
      </button>
    </li>
    ...
  </ul>
  ```
</CodeExample>

### is-active

Adding the `is-active` class will provide visual indication that the action is currently in an active state.

<CodeExample>
  <Fragment slot="output">
    <ul class="menu menu_inline">
      <li class="menu__item">
        <button class="menu__action is-active" disabled>One</button>
      </li>
      <li class="menu__item">
        <button class="menu__action">Two</button>
      </li>
      <li class="menu__item">
        <button class="menu__action">Three</button>
      </li>
    </ul>
  </Fragment>
  ```html
  <ul class="menu menu_inline">
    <li class="menu__item">
      <button class="menu__action is-active" disabled>
        ...
      </button>
    </li>
    ...
  </ul>
  ```
</CodeExample>

### is-disabled

Adding the boolean `disabled` attribute or `is-disabled` class will provide visual indication that the user should not be able to interact with the action.

<CodeExample>
  <Fragment slot="output">
    <ul class="menu menu_inline">
      <li class="menu__item">
        <button class="menu__action is-disabled" disabled>One</button>
      </li>
      <li class="menu__item">
        <button class="menu__action">Two</button>
      </li>
      <li class="menu__item">
        <button class="menu__action">Three</button>
      </li>
    </ul>
  </Fragment>
  ```html
  <ul class="menu menu_inline">
    <li class="menu__item">
      <button class="menu__action is-disabled" disabled>
        ...
      </button>
    </li>
    ...
  </ul>
  ```
</CodeExample>

## menu_inline

Used to apply horizontal menu styles. This is typically used for short menus or toolbars where vertical space can be saved.

<CodeExample>
  <Fragment slot="output">
    <Menu name="crud" classes="menu_inline" />
  </Fragment>
  ```html
  <ul class="menu menu_inline">...</ul>
  ```
</CodeExample>

## menu_full

Used to span a horizontal menu to fill the full width of its container. This modifier is meant to be paired with the `menu_inline` modifier as the default styles of a vertical menu already fill the full width of their container.

<CodeExample>
  <div slot="output" class="gap-y">
    <Menu name="crud" classes="menu_inline menu_full" />
  </div>
  ```html
  <ul class="menu menu_inline menu_full">...</ul>
  ```
</CodeExample>

## menu_size_[value]

Adjust the size of a menu by increasing or decreasing its padding and font-size. By default, the menu scale will provide an action height of 30px (small `menu_size_sm`), 40px (default) and 50px (large `menu_size_lg`).

<CodeExample>
  <div slot="output" class="gap-y">
    <Menu name="crud" classes="menu_inline menu_size_sm" />
    <Menu name="crud" classes="menu_inline menu_size_lg" />
  </div>
  ```html
  <ul class="menu menu_size_sm">
    ...
  </ul>

  <ul class="menu menu_size_lg">
    ...
  </ul>
  ```
</CodeExample>

**Available variants**

- `menu_size_sm`
- `menu_size_lg`

## Customization

Menus are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="menu/src/_root.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";
  @use "./variables" as var;

  :root {
    @include core.css("menu-size", var.$size);
    @include core.css("menu-padding", var.$padding);
    @include core.css("menu-padding-x", core.get-side(var.$padding, top));
    @include core.css("menu-gap", var.$gap);
    @include core.css("menu-border-radius", var.$border-radius);
    @include core.css("menu-background", var.$background);
    @include core.css("menu-background-hover", var.$background-hover);
    @include core.css("menu-background-focus", var.$background-focus);
    @include core.css("menu-background-active", var.$background-active);
    @include core.css("menu-foreground", var.$foreground);
    @include core.css("menu-foreground-hover", var.$foreground-hover);
    @include core.css("menu-foreground-focus", var.$foreground-focus);
    @include core.css("menu-foreground-active", var.$foreground-active);
    @include core.css("menu-outline-focus", var.$outline-focus);
    @include core.css("menu-font-size", var.$font-size);
    @include core.css("menu-line-height", var.$line-height);

    @include core.css("menu-action-gap", var.$action-gap);

    @include core.css("menu-sep-size", var.$sep-size);
    @include core.css("menu-sep-gap", var.$sep-gap);
    @include core.css("menu-sep-background", var.$sep-background);

    @include core.css("menu-active-background", var.$active-background);
    @include core.css("menu-active-foreground", var.$active-foreground);

    @include core.css("menu-disabled-background", var.$disabled-background);
    @include core.css("menu-disabled-foreground", var.$disabled-foreground);
  }

  @include theme.output("menu");
  ```
</DocBlock>

<DocBlock name="SCSS Variables" src="menu/src/_variables.scss">

  ```scss
  @use "@vrembem/core/palette";
  @use "@vrembem/core/theme";
  @use "@vrembem/core";

  $size: core.$form-control-size !default;
  $padding: core.$form-control-padding !default;
  $gap: 1px !default;
  $border-radius: core.$border-radius !default;
  $background: null !default;
  $background-hover: null !default;
  $background-focus: null !default;
  $background-active: null !default;
  $foreground: null !default;
  $foreground-hover: null !default;
  $foreground-focus: null !default;
  $foreground-active: null !default;
  $outline-focus: 2px solid theme.get("focus-visible") !default;
  $font-size: 1em !default;
  $line-height: 1.5 !default;

  // menu__action
  $action-gap: 0.5em !default;

  // menu__sep
  $sep-size: 1px !default;
  $sep-gap: 0.5em !default;
  $sep-background: theme.get("border-color") !default;

  // is-active
  $active-background: none !default;
  $active-foreground: palette.get("primary") !default;

  // is-disabled
  $disabled-background: none !default;
  $disabled-foreground: theme.get("foreground-lighter") !default;

  // menu_size_sm
  $size-sm: core.$form-control-size-sm !default;
  $size-sm-padding: core.$form-control-padding-sm !default;
  $size-sm-font-size: core.$font-size-sm !default;
  $size-sm-line-height: core.css("line-height")-sm !default;

  // menu_size_sm
  $size-lg: core.$form-control-size-lg !default;
  $size-lg-padding: core.$form-control-padding-lg !default;
  $size-lg-font-size: core.$font-size-lg !default;
  $size-lg-line-height: core.css("line-height")-lg !default;

  // Theme maps
  $theme-shared: (
    "background": none,
    "background-hover": theme.get("background-hover"),
    "background-focus": theme.get("background-focus"),
    "background-active": theme.get("background-active"),
    "foreground": theme.get("foreground"),
  ) !default;

  @include theme.set("light", "menu", $theme-shared);
  @include theme.set("dark", "menu", $theme-shared);
  ```
</DocBlock>
