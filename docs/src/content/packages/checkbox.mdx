# Checkbox

Checkboxes allow the user to select multiple options from a set.

```sh
npm install @vrembem/checkbox
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/checkbox";
  ```
</CodeExample>

## Config options

Checkbox comes with the following configuration options that can be set using either the [config module](/packages/core/config#using-config-module) or [Sass with clause](/packages/core/config#using-with-clause) methods.

```scss
$config: (
  // Map for outputting checkbox_size_[key] modifier variants
  // @type map
  "checkbox-size": (
    "breakpoints": true,
    "default": css.get("form-control-size"),
    "sm": css.get("form-control-size-sm"),
    "md": css.get("form-control-size"),
    "lg": css.get("form-control-size-lg")
  )
);
```

## Basic usage

Checkboxes are composed using a set of `<span>` elements alongside the native `<input type="checkbox">` element which should be given the `checkbox__native` class and come before the remaining presentational `<span>` elements.

<CodeExample tabs>
  <Fragment slot="output">
    <span class="checkbox">
      <input class="checkbox__native" type="checkbox" checked />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
    <span class="checkbox">
      <input class="checkbox__native" type="checkbox" aria-checked="mixed" />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
    <span class="checkbox">
      <input class="checkbox__native" type="checkbox" />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
  </Fragment>
  <Fragment slot="tab-html">
    ```html
    <span class="checkbox">
      <input class="checkbox__native" type="checkbox" />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
    ```
  </Fragment>
  <Fragment slot="tab-js">
    ```js
    // Get checkboxes with "mixed" status
    const checkboxes = document.querySelectorAll(
      "input[type='checkbox'][aria-checked='mixed']"
    );

    // Apply the indeterminate state
    checkboxes.forEach((checkbox) => checkbox.indeterminate = true);
    ```
  </Fragment>
</CodeExample>

Indeterminate checkboxes need to be set in JavaScript though styling is provided by Vrembem for the indeterminate states (see example above). Review [MDN documentation](https://developer.mozilla.org/en-US/docs/Web/CSS/:indeterminate) for more details.

## checkbox_size_[value]

Adjust the size of a checkbox by increasing or decreasing its font-size. These modifiers are generated using key/value pairs of the `checkbox-size` config options map.

<CodeExample>
  <Fragment slot="output">
    <span class="checkbox checkbox_size_sm">
      <input class="checkbox__native" type="checkbox" checked />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
    <span class="checkbox">
      <input class="checkbox__native" type="checkbox" checked />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
    <span class="checkbox checkbox_size_lg">
      <input class="checkbox__native" type="checkbox" checked />
      <span class="checkbox__background">
        <span class="checkbox__box">
          <span class="checkbox__icon"></span>
        </span>
      </span>
    </span>
  </Fragment>
  ```html
  <span class="checkbox checkbox_size_sm">
    ...
  </span>

  <span class="checkbox checkbox_size_lg">
    ...
  </span>
  ```
</CodeExample>

**Available variants**

- `checkbox_size_sm`
- `checkbox_size_lg`

### Breakpoint variants

These modifiers come with media breakpoint variants. This allows changing styles based on a specific breakpoint key in the [`breakpoints`](/packages/core#config-options) config options map. Breakpoint variants output can be toggled using the `breakpoints` key in the `checkbox-size` config options map (default: `true`).

<CodeExample>
  <div slot="output" class="spacing">
    <p>Current view-port width: <code data-vp-width>...</code></p>
    <label class="flex flex_inline flex_gap_sm align-items-center">
      <span class="checkbox checkbox_size_sm lg:checkbox_size_md xl:checkbox_size_lg">
        <input class="checkbox__native" type="checkbox" checked />
        <span class="checkbox__background">
          <span class="checkbox__box">
            <span class="checkbox__icon"></span>
          </span>
        </span>
      </span>
      <span>Checkbox</span>
      <span class="badge lg:display-none">size_sm</span>
      <span class="badge display-none lg:display-block xl:display-none">size_md</span>
      <span class="badge display-none xl:display-block">size_lg</span>
    </label>
  </div>
  ```html
  <span class="checkbox checkbox_size_sm lg:checkbox_size_md xl:checkbox_size_lg">
    ...
  </span>
  ```
</CodeExample>

**Available variants**

- `xs:checkbox_size_[key]`
- `sm:checkbox_size_[key]`
- `md:checkbox_size_[key]`
- `lg:checkbox_size_[key]`
- `xl:checkbox_size_[key]`
