# Input

A component for displaying form input elements.

```sh
npm install @vrembem/input
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/input";
  ```
</CodeExample>

## Basic usage

The most basic implementation of the input component consists of the `input` class applied to an `<input>` element.

<CodeExample>
  <Fragment slot="output">
    <input class="input" type="text" placeholder="Example input..." />
  </Fragment>
  ```html
  <input class="input" type="text" placeholder="Example input..." />
  ```
</CodeExample>

### Disabled

Adding the boolean `disabled` attribute to the input will provide visual indication that the input is not available for use.

<CodeExample>
  <Fragment slot="output">
    <input class="input" type="text" disabled />
  </Fragment>
  ```html
  <input class="input" type="text" disabled />
  ```
</CodeExample>

### Readonly

Adding the boolean `readonly` attribute to the input will provide visual indication that the user should not be able to edit the value of the input.

<CodeExample>
  <Fragment slot="output">
    <input class="input" type="text" value="Readonly example" readonly />
  </Fragment>
  ```html
  <input class="input" type="text" readonly />
  ```
</CodeExample>

## input_auto

Sets the width of an input to `auto` instead of the base component width of `100%`.

<CodeExample>
  <Fragment slot="output">
    <input class="input input_auto" type="text" />
  </Fragment>
  ```html
  <input class="input input_auto" type="text" />
  ```
</CodeExample>

## input_size_[value]

Adjust the size of an input by increasing or decreasing its padding and font-size. By default, the input scale will provide an input height of 30px (small `input_size_sm`), 40px (default) and 50px (large `input_size_lg`).

<CodeExample>
  <div slot="output" class="gap-y">
    <input class="input input_size_sm" type="text" />
    <input class="input input_size_lg" type="text" />
  </div>
  ```html
  <input class="input input_size_sm" type="text" />
  <input class="input input_size_lg" type="text" />
  ```
</CodeExample>

**Available variants**

- `input_size_sm`
- `input_size_lg`

## input_state_[value]

Adds styles for changing the look and feel of an input to better reflect the current state.

<CodeExample>
  <div slot="output" class="gap-y">
    <input class="input input_state_success" type="text" />
    <input class="input input_state_error" type="text" />
  </div>
  ```html
  <input class="input input_state_success" type="text" />
  <input class="input input_state_danger" type="text" />
  ```
</CodeExample>

**Available variants**

- `input_state_success`
- `input_state_error`

## input_type_[value]

Adds unique styles for various form input types. These form controls share styles with the basic form input such as [`<select>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/select) and [`<textarea>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea) HTML elements.

<CodeExample resettable>
  <div slot="output" class="gap-y">
    <select class="input input_type_select" name="">
      <option value="">Select one</option>
      <option value="1">Option 1</option>
      <option value="2">Option 2</option>
      <option value="3">Option 3</option>
    </select>
    <textarea class="input input_type_textarea" rows="3"></textarea>
  </div>
  ```html
  <select class="input input_type_select" name="">
    <option value="">Select one</option>
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
    <option value="3">Option 3</option>
  </select>

  <textarea class="input input_type_textarea" rows="3"></textarea>
  ```
</CodeExample>

**Available variants**

- `input_type_select`
- `input_type_textarea`

## Customization

Inputs are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="input/src/_root.scss">

  ```scss
  @use "@vrembem/core";
  @use "@vrembem/core/theme";
  @use "./variables" as var;

  :root {
    @include core.css("input-size", var.$size);
    @include core.css("input-padding", var.$padding);
    @include core.css("input-background", var.$background);
    @include core.css("input-background-hover", var.$background-hover);
    @include core.css("input-background-focus", var.$background-focus);
    @include core.css("input-foreground", var.$foreground);
    @include core.css("input-foreground-hover", var.$foreground-hover);
    @include core.css("input-foreground-focus", var.$foreground-focus);
    @include core.css("input-foreground-placeholder", var.$foreground-placeholder);
    @include core.css("input-border-color", var.$border-color);
    @include core.css("input-border-color-hover", var.$border-color-hover);
    @include core.css("input-border-color-focus", var.$border-color-focus);
    @include core.css("input-border-width", var.$border-width);
    @include core.css("input-border-radius", var.$border-radius);
    @include core.css("input-box-shadow", var.$box-shadow);
    @include core.css("input-box-shadow-focus", var.$box-shadow-focus);
    @include core.css("input-box-shadow-inset", var.$box-shadow-inset);
    @include core.css("input-box-shadow-inset-focus", var.$box-shadow-inset-focus);
    @include core.css("input-font-size", var.$font-size);
    @include core.css("input-line-height", var.$line-height);
    @include core.css("input-outline", var.$outline);
    @include core.css("input-transition-property", var.$transition-property);
    @include core.css("input-transition-duration", var.$transition-duration);
    @include core.css("input-transition-timing-function", var.$transition-timing-function);
    @include core.css("input-background-disabled", var.$background-disabled);
    @include core.css("input-background-readonly", var.$background-readonly);
  }

  @include theme.output("input");
  ```
</DocBlock>

<DocBlock name="SCSS Variables" src="input/src/_variables.scss">

  ```scss
  @use "sass:list";
  @use "@vrembem/core";
  @use "@vrembem/core/palette";
  @use "@vrembem/core/theme";

  $size: core.$form-control-size !default;
  $padding: calc(#{list.nth(core.$form-control-padding, 1)} - 1px) !default;
  $background: null !default;
  $background-hover: null !default;
  $background-focus: null !default;
  $foreground: null !default;
  $foreground-hover: null !default;
  $foreground-focus: null !default;
  $foreground-placeholder: null !default;
  $border-color: null !default;
  $border-color-hover: null !default;
  $border-color-focus: null !default;
  $border-width: 1px !default;
  $border-radius: core.$border-radius !default;
  $box-shadow: palette.get("primary", 50%, 0%) !default;
  $box-shadow-focus: palette.get("primary", 50%, 50%) !default;
  $box-shadow-inset: rgb(0 0 0 / 10%);
  $box-shadow-inset-focus: rgb(0 0 0 / 0%);
  $font-size: core.$font-size !default;
  $line-height: core.css("line-height") !default;
  $outline: none !default;

  // Transition
  $transition-property: box-shadow, outline, outline-offset !default;
  $transition-duration: core.$transition-duration-short !default;
  $transition-timing-function: core.$transition-timing-function !default;

  // Disabled
  $background-disabled: theme.get("background-darker") !default;

  // Readonly
  $background-readonly: theme.get("background-dark") !default;

  // input_size_sm
  $size-sm: core.$form-control-size-sm !default;
  $size-sm-padding: calc(#{list.nth(core.$form-control-padding-sm, 1)} - 1px) list.nth(core.$form-control-padding-sm, 2) !default;
  $size-sm-font-size: core.$font-size-sm !default;
  $size-sm-line-height: core.css("line-height")-sm !default;

  // input_size_lg
  $size-lg: core.$form-control-size-lg !default;
  $size-lg-padding: calc(#{list.nth(core.$form-control-padding-lg, 1)} - 1px) !default;
  $size-lg-font-size: core.$font-size-lg !default;
  $size-lg-line-height: core.css("line-height")-lg !default;

  // input_type_[value]
  $type-select-icon-color: palette.get("neutral", "seed") !default;
  $type-select-icon: 'data:image/svg+xml;utf8,<svg xmlns="http://www.w3.org/2000/svg" width="22" height="12" fill="#{core.encodecolor($type-select-icon-color)}"><polyline points="0 8 3.5 12 7 8"></polyline><polyline points="0 4 3.5 0 7 4"></polyline></svg>' !default;
  $type-textarea-min-width: 100% !default;

  // Theme maps
  $theme-light: (
    "background": theme.get("background"),
    "foreground": theme.get("foreground"),
    "foreground-placeholder": theme.get("foreground-lighter"),
    "border-color": theme.get("border-color-dark"),
    "border-color-hover": theme.get("border-color-darker"),
    "border-color-focus": palette.get("primary")
  ) !default;

  $theme-dark: (
    "background": theme.get("background"),
    "foreground": theme.get("foreground"),
    "foreground-placeholder": theme.get("foreground-lighter"),
    "border-color": theme.get("border-color-dark"),
    "border-color-hover": theme.get("border-color-darker"),
    "border-color-focus": palette.get("primary", 60)
  ) !default;

  @include theme.set("light", "input", $theme-light);
  @include theme.set("dark", "input", $theme-dark);
  ```
