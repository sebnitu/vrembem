# Card

The card component provides a flexible and highly composable content container.

```sh
npm install @vrembem/card
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/card";
  ```
</CodeExample>

## Basic usage

Cards are a highly composable container component that comes with a variety of elements. The three most basic are:

- `card__body`
- `card__image`
- `card__title`

<CodeExample>
  <div slot="output">
    <div class="card">
      <img class="card__image" src="https://picsum.photos/id/335/600/300" />
      <div class="card__body">
        <h2 class="card__title">Card Title</h2>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
    </div>
  </div>
  ```html
  <div class="card">
    <img class="card__image" src="..." />
    <div class="card__body">
      <h2 class="card__title">...</h2>
      ...
    </div>
  </div>
  ```
</CodeExample>

For creating more distinct sections in a card, header and footer elements can be used. It's also possible to have multiple `card__body` elements in your cards as needed.

- `card__header`
- `card__footer`

<CodeExample>
  <div slot="output" class="spacing">
    <div class="card">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body spacing">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
        <p class="foreground-lighter flex justify-content-between">
          <span>April 9th, 2024</span>
          <span>30 comments</span>
        </p>
      </div>
      <div class="card__footer flex justify-content-between">
        <button class="button button_color_primary">Read More</button>
        <button class="button button_color_secondary button_icon">
          <Icon name="heart" />
        </button>
      </div>
    </div>
  </div>
  ```html
  <div class="card">
    <div class="card__header">
      <h2 class="card__title">...</h2>
    </div>
    <div class="card__body">
      ...
    </div>
    <div class="card__footer">
      ...
    </div>
  </div>
  ```
</CodeExample>

Card screens and backgrounds are displayed behind other card elements. These can be paired with the `vb-theme-light` or `vb-theme-dark` theme classes to create a light or dark card theme.

- `card__screen`
- `card__background`

<CodeExample>
  <div slot="output" class="spacing">
    <div class="card vb-theme-light">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
      <img src="https://picsum.photos/id/976/600/300" class="card__background" />
      <div class="card__screen"></div>
    </div>
    <div class="card vb-theme-dark">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
      <img src="https://picsum.photos/id/976/600/300" class="card__background" />
      <div class="card__screen"></div>
    </div>
  </div>
  ```html
  <div class="card">
    ...
    <img src="..." class="card__background" />
    <div class="card__screen"></div>
  </div>
  ```
</CodeExample>

When creating a card using the `<a>` element, additional styles are applied to make the card more visually interactive with hover and focus styles.

<CodeExample>
  <div slot="output" class="spacing">
    <a href="#" class="card">
      <img class="card__image" src="https://picsum.photos/id/106/600/300" />
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body spacing">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci id venenatis nisi quam non turpis... <strong>Continue Reading</strong></p>
      </div>
    </a>
  </div>
  ```html
  <a href="#" class="card">
    ...
  </a>
  ```
</CodeExample>

## Customization

Cards are primarily customized using CSS variables but can also be modified using Sass variables. All CSS variables are given the Vrembem vendor prefix `"vb-"`. This can be changed by setting the `$prefix-variable` variable via the `core` module.

<DocBlock name="CSS Variables" src="card/src/_root.scss">

```scss
:root {
  --vb-card-padding: 1.25em;
  --vb-card-border-radius: var(--vb-border-radius);
  --vb-card-box-shadow: var(--vb-box-shadow-1);
  --vb-card-link-box-shadow: var(--vb-box-shadow-2);
  --vb-card-link-box-shadow-hover: var(--vb-box-shadow-3);
  --vb-card-link-offset: -0.25em;
  --vb-card-title-font-size: var(--vb-font-size-lg);
  --vb-card-title-line-height: var(--vb-line-height-lg);
  --vb-card-title-font-weight: var(--vb-font-weight-semi-bold);
  --vb-card-transition-property: background-color, border-color, box-shadow, transform;
  --vb-card-transition-duration: var(--vb-transition-duration);
  --vb-card-transition-timing-function: var(--vb-transition-timing-function);
}

:root, .vb-theme-root, .vb-theme-light {
  --vb-card-background: var(--vb-background);
  --vb-card-foreground: var(--vb-foreground-light);
  --vb-card-border: 1px solid var(--vb-border-color);
  --vb-card-sep-border: 1px solid var(--vb-border-color);
  --vb-card-title-color: var(--vb-foreground);
  --vb-card-screen-background: var(--vb-background);
  --vb-card-screen-opacity: 0.9;
}

@media (prefers-color-scheme: dark) {
  :root, .vb-theme-root {
    --vb-card-background: var(--vb-background-darker);
    --vb-card-foreground: var(--vb-foreground-light);
    --vb-card-border: 1px solid var(--vb-border-color);
    --vb-card-sep-border: 1px solid var(--vb-border-color);
    --vb-card-title-color: var(--vb-foreground);
    --vb-card-screen-background: var(--vb-background-darker);
    --vb-card-screen-opacity: 0.8;
  }
}

.vb-theme-dark {
  --vb-card-background: var(--vb-background-darker);
  --vb-card-foreground: var(--vb-foreground-light);
  --vb-card-border: 1px solid var(--vb-border-color);
  --vb-card-sep-border: 1px solid var(--vb-border-color);
  --vb-card-title-color: var(--vb-foreground);
  --vb-card-screen-background: var(--vb-background-darker);
  --vb-card-screen-opacity: 0.8;
}
```
</DocBlock>

<DocBlock name="SCSS Variables" src="card/src/_variables.scss">

```scss
@use "@vrembem/core";
@use "@vrembem/core/css";
@use "@vrembem/core/theme";

$padding: 1.25em !default;
$background: null !default;
$foreground: null !default;
$border: null !default;
$sep-border: null !default;
$border-radius: css.get("border-radius") !default;
$box-shadow: css.get("box-shadow-1") !default;

// Screen
$screen-background: null !default;
$screen-opacity: null !default;

// Link
$link-box-shadow: css.get("box-shadow-2") !default;
$link-box-shadow-hover: css.get("box-shadow-3") !default;
$link-offset: -0.25em !default;

// Title
$title-color: null !default;
$title-font-size: css.get("font-size-lg") !default;
$title-line-height: css.get("line-height-lg") !default;
$title-font-weight: css.get("font-weight-semi-bold") !default;

// Transition
$transition-property: background-color, border-color, box-shadow, transform !default;
$transition-duration: css.get("transition", "duration") !default;
$transition-timing-function: css.get("transition", "timing-function") !default;

// Custom properties
@include css.set("card", (
  "padding": $padding,
  "background": $background,
  "foreground": $foreground,
  "border": $border,
  "sep-border": $sep-border,
  "border-radius": $border-radius,
  "box-shadow": $box-shadow,
  "screen-background": $screen-background,
  "screen-opacity": $screen-opacity,
  "link-box-shadow": $link-box-shadow,
  "link-box-shadow-hover": $link-box-shadow-hover,
  "link-offset": $link-offset,
  "title-color": $title-color,
  "title-font-size": $title-font-size,
  "title-line-height": $title-line-height,
  "title-font-weight": $title-font-weight,
  "transition-property": $transition-property,
  "transition-duration": $transition-duration,
  "transition-timing-function": $transition-timing-function
));

// Themes
$theme-light: (
  "background": theme.get("background"),
  "foreground": theme.get("foreground-light"),
  "border": 1px solid theme.get("border-color"),
  "sep-border": 1px solid theme.get("border-color"),
  "title-color": theme.get("foreground"),
  "screen-background": theme.get("background"),
  "screen-opacity": 0.9
) !default;

$theme-dark: (
  "background": theme.get("background-darker"),
  "foreground": theme.get("foreground-light"),
  "border": 1px solid theme.get("border-color"),
  "sep-border": 1px solid theme.get("border-color"),
  "title-color": theme.get("foreground"),
  "screen-background": theme.get("background-darker"),
  "screen-opacity": 0.8
) !default;

@include theme.set("light", "card", $theme-light);
@include theme.set("dark", "card", $theme-dark);
```
