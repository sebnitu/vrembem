# Switch

Switches are a binary form element used to toggle between two options.

```sh
npm install @vrembem/switch
```

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/switch";
  ```
</CodeExample>

## Config options

Switch comes with the following configuration options that can be set using either the [config module](/packages/core/config#using-config-module) or [Sass with clause](/packages/core/config#using-with-clause) methods.

```scss
$config: (
  // Map for outputting switch_size_[key] modifier variants
  // @type map
  "switch-size": (
    "breakpoints": true,
    "default": css.get("form-control-size"),
    "sm": css.get("form-control-size-sm"),
    "md": css.get("form-control-size"),
    "lg": css.get("form-control-size-lg")
  )
);
```

## Basic usage

Switch form controls are composed using a set of `<span>` elements alongside the native `<input type="checkbox">` element which should be given the `switch__native` class and come before the remaining presentational `<span>` elements.

<CodeExample>
  <Fragment slot="output">
    <span class="switch">
      <input type="checkbox" class="switch__native" checked />
      <span class="switch__background">
        <span class="switch__track">
          <span class="switch__thumb"></span>
        </span>
      </span>
    </span>
    <span class="switch">
      <input type="checkbox" class="switch__native" />
      <span class="switch__background">
        <span class="switch__track">
          <span class="switch__thumb"></span>
        </span>
      </span>
    </span>
  </Fragment>
  ```html
  <span class="switch">
    <input type="checkbox" class="switch__native">
    <span class="switch__background">
      <span class="switch__track">
        <span class="switch__thumb"></span>
      </span>
    </span>
  </span>
  ```
</CodeExample>

## switch_size_[value]

Adjust the size of a switch by increasing or decreasing its font-size. These modifiers are generated using key/value pairs of the `switch-size` config options map.

<CodeExample>
  <Fragment slot="output">
    <span class="switch switch_size_sm">
      <input type="checkbox" class="switch__native" />
      <span class="switch__background">
        <span class="switch__track">
          <span class="switch__thumb"></span>
        </span>
      </span>
    </span>
    <span class="switch">
      <input type="checkbox" class="switch__native" />
      <span class="switch__background">
        <span class="switch__track">
          <span class="switch__thumb"></span>
        </span>
      </span>
    </span>
    <span class="switch switch_size_lg">
      <input type="checkbox" class="switch__native" />
      <span class="switch__background">
        <span class="switch__track">
          <span class="switch__thumb"></span>
        </span>
      </span>
    </span>
  </Fragment>
  ```html
  <span class="switch switch_size_sm">
    ...
  </span>

  <span class="switch switch_size_lg">
    ...
  </span>
  ```
</CodeExample>

**Available variants**

- `switch_size_sm`
- `switch_size_lg`

### Breakpoint variants

These modifiers come with media breakpoint variants. This allows changing styles based on a specific breakpoint key in the [`breakpoints`](/packages/core#config-options) config options map. Breakpoint variants output can be toggled using the `breakpoints` key in the `switch-size` config options map (default: `true`).

<CodeExample>
  <div slot="output" class="spacing">
    <p>Current view-port width: <code data-vp-width>...</code></p>
    <label class="flex flex_inline flex_gap_sm align-items-center">
      <span class="switch switch_size_sm lg:switch_size_md xl:switch_size_lg">
        <input type="checkbox" class="switch__native" />
        <span class="switch__background">
          <span class="switch__track">
            <span class="switch__thumb"></span>
          </span>
        </span>
      </span>
      <span>Switch</span>
      <span class="badge lg:display-none">size_sm</span>
      <span class="badge display-none lg:display-block xl:display-none">size_md</span>
      <span class="badge display-none xl:display-block">size_lg</span>
    </label>
  </div>
  ```html
  <span class="switch switch_size_sm lg:switch_size_md xl:switch_size_lg">
    ...
  </span>
  ```
</CodeExample>

**Available variants**

- `xs:switch_size_[key]`
- `sm:switch_size_[key]`
- `md:switch_size_[key]`
- `lg:switch_size_[key]`
- `xl:switch_size_[key]`
