## Options

Popover is built as a Sass module. Sass options are used to control CSS output, while design token CSS variables control the styling of the component. Below are the available configuration options.

### Sass

Popover comes with the following Sass configuration options that can be set using either the [config module](/reference/sass-modules/config#using-config-module) or [Sass `with` clause](/reference/sass-modules/config#using-with-clause) methods.

<CodeExample lang="scss">
```scss
@use "@vrembem/popover" with (
  $config: (
    // Map for outputting popover--width-[key] modifier variants
    // @type map
    "popover-width": (
      "default": 14rem,
      "auto": "fit-content",
      "sm": 10rem,
      "lg": 18rem
    )
  )
);
```
</CodeExample>

## Basic usage

Vrembem popovers are powered by the native browser popover API and CSS anchor positions. This allows popovers to function without any need for additional JavaScript. For a basic popover, just apply the following attributes along with the `.popover` class for styling.

<CodeExample>
  <Fragment slot="output">
    <button class="button button--color-primary" popovertarget="popover-1">Popover</button>
    <div class="popover" id="popover-1" popover>
      <Menu name="format-text" />
    </div>
  </Fragment>
  ```html
  <button popovertarget="unique-id">...</button>
  <div class="popover" id="unique-id" popover>
    ...
  </div>
  ```
</CodeExample>

## popover--pos-[key]

A modifier to set the preferred position of a popover relative to its anchor. If not enough space is available in the preferred position, the popover is flipped automatically.

      <button class="button" popovertarget="popover-right-1">Right</button>
      <div id="popover-top-1" class="popover popover--pos-top" popover><Menu name="edit-shortcuts" /></div>
      <div id="popover-bottom-1" class="popover popover--pos-bottom" popover><Menu name="edit-shortcuts" /></div>

      <div id="popover-right-1" class="popover popover--pos-right" popover><Menu name="edit-shortcuts" /></div>
    </div>
    <div class="width-fit-content margin-x-auto">
      <button class="button" popovertarget="popover-top-start-1">Top Start</button>
      <button class="button" popovertarget="popover-bottom-start-1">Bottom Start</button>
      <button class="button" popovertarget="popover-left-start-1">Left Start</button>

## popover--arrow

Modifier that adds an arrow to the popover element. When added along with a position modifier, using `.popover--arrow` will infer the arrow direction. To set an explicit arrow direction, provide a modifier value `.popover--arrow-[key]`.

<CodeExample>
  <Fragment slot="output">
    <div class="width-fit-content margin-x-auto">
      <button class="button" popovertarget="popover-arrow-1a" interestfor="popover-arrow-1b">Arrow 1</button>
      <div id="popover-arrow-1a" class="popover popover--arrow-top" popover><Menu name="edit-shortcuts" /></div>
      <div id="popover-arrow-1b" class="popover popover--arrow-bottom popover--tooltip" popover="hint">Tooltip</div>

      <button class="button" popovertarget="popover-arrow-2a" interestfor="popover-arrow-2b">Arrow 2</button>
      <div id="popover-arrow-2a" class="popover popover--arrow popover--pos-top" popover><Menu name="edit-shortcuts" /></div>
      <div id="popover-arrow-2b" class="popover popover--arrow popover--pos-bottom popover--tooltip" popover="hint">Tooltip</div>

      <button class="button" popovertarget="popover-arrow-3a" interestfor="popover-arrow-3b">Arrow 3</button>
      <div id="popover-arrow-3a" class="popover popover--arrow popover--pos-left" popover><Menu name="edit-shortcuts" /></div>
      <div id="popover-arrow-3b" class="popover popover--arrow popover--pos-right popover--tooltip" popover="hint">Tooltip</div>

      <button class="button" popovertarget="popover-arrow-4a" interestfor="popover-arrow-4b">Arrow 4</button>
      <div id="popover-arrow-4a" class="popover popover--arrow popover--pos-right" popover><Menu name="edit-shortcuts" /></div>
      <div id="popover-arrow-4b" class="popover popover--arrow popover--pos-left popover--tooltip" popover="hint">Tooltip</div>
    </div>
  </Fragment>
  ```html
  <!-- Apply the arrow modifier with a position modifier -->
  <div id="unique-id" class="popover popover--arrow popover--pos-top" popover="hint">
    ...
  </div>

  <!-- Set the arrow direction if no position modifier is set -->
  <div id="unique-id" class="popover popover--arrow-bottom" popover="hint">
    ...
  </div>
  ```
</CodeExample>

**Available variants**

- `popover--arrow`
- `popover--arrow-top`
- `popover--arrow-bottom`
- `popover--arrow-left`
- `popover--arrow-right`

## popover--tooltip

To create a tooltip popover, use the interest invoker API along with the `.popover--tooltip` modifier class for styling. Tooltips appear above their target element by default. Use position modifiers to adjust the position of tooltips.

<CodeExample>
  <Fragment slot="output">
    <div class="flex">
      <button type="button" class="link" interestfor="popover-tooltip-1">HTML</button>
      <button type="button" class="link" interestfor="popover-tooltip-2">CSS</button>
      <button type="button" class="link" interestfor="popover-tooltip-3">JS</button>
    </div>
    <div id="popover-tooltip-1" class="popover popover--tooltip popover--arrow-bottom" popover="hint">
      Hypertext Markup Language
    </div>
    <div id="popover-tooltip-2" class="popover popover--tooltip popover--arrow-bottom" popover="hint">
      Cascading Style Sheets
    </div>
    <div id="popover-tooltip-3" class="popover popover--tooltip popover--arrow-bottom" popover="hint">
      JavaScript
    </div>
  </Fragment>
  ```html
  <button interestfor="unique-id">HTML</button>
  <div id="unique-id" popover="hint" class="popover popover--tooltip popover--arrow">
    Hypertext Markup Language
  </div>
  ```
</CodeExample>

Popover and interest invoker APIs can be applied to the same anchor. Use the `popovertarget` and `interestfor` attributes with the values of their corresponding popover IDs.

<CodeExample>
  <Fragment slot="output">
    <button type="button" class="button button--icon" popovertarget="popover-multi-1" interestfor="popover-multi-2">
      <Icon name="menu" />
    </button>
    <div id="popover-multi-1" class="popover popover--arrow-top" popover><Menu name="edit-shortcuts" /></div>
    <div id="popover-multi-2" class="popover popover--tooltip popover--arrow-bottom" popover="hint">Menu</div>
  </Fragment>
  ```html
  <button popovertarget="unique-id-1" interestfor="unique-id-2">
    ...
  </button>
  <div id="unique-id-1" popover class="popover">
    ...
  </div>
  <div id="unique-id-2" popover="hint" class="popover popover--tooltip">
    ...
  </div>
  ```
</CodeExample>

## popover--width-[key]

A modifier for adjusting the width of a popover. These modifiers are generated using key/value pairs of the `popover-width` config options map.

<CodeExample>
  <div slot="output" class="flex">
    <button type="button" class="button" popovertarget="popover-7">Auto</button>
    <div id="popover-7" class="popover popover--width-auto" popover>
      <Menu name="edit-shortcuts" />
    </div>
    <button type="button" class="button" popovertarget="popover-8">Small</button>
    <div id="popover-8" class="popover popover--width-sm" popover>
      <Menu name="edit-shortcuts" />
    </div>
    <button type="button" class="button" popovertarget="popover-9">Default</button>
    <div id="popover-9" class="popover" popover>
      <Menu name="edit-shortcuts" />
    </div>
    <button type="button" class="button" popovertarget="popover-10">Large</button>
    <div id="popover-10" class="popover popover--width-lg" popover>
      <Menu name="edit-shortcuts" />
    </div>
  </div>
  ```html
  <div id="unique-id" popover class="popover popover--width-sm">
    ...
  </div>
  ```
</CodeExample>

**Available variants**

- `popover--width-auto`
- `popover--width-sm`
- `popover--width-lg`
