# Packages

Vrembem is a collection of packages that consist of a Sass and JavaScript modules. To install and use the entire library, use the `vrembem` package:

```sh
npm install vrembem
```

<CodeExample lang="scss">
  ```scss
  @use "vrembem";
  ```
</CodeExample>

<CodeExample lang="js">
  ```js
  import * as vb from "vrembem";
  ```
</CodeExample>
