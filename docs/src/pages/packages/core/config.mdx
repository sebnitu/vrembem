This module is used to setup the global configuration options. This is used across all components to set things like prefixes, BEM naming syntax and default themes.

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/core/config";
  ```
</CodeExample>

## Basic usage

- `config.get()`
- `config.set()`
- `config.remove()`
- `config.log()`
