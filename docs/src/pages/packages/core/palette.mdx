Defines the design system's color palette. This takes in a map of key and color seed values and builds a variety of usable color values for your project.

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/core/palette";
  ```
</CodeExample>

## Basic usage

- `palette.build()`
- `palette.get()`
- `palette.set()`
- `palette.remove()`
