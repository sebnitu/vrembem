Defines the design system's color palette. Using a map of color seeds, it builds a variety of color palette custom properties under the `palette` namespace.

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/core/palette";
  ```
</CodeExample>

## Basic usage

This module extends `core/css` by creating a custom properties color palette map under the `palette` namespace. It uses a seeds map containing color names and values, a list of lightness values to generate variations, and a default lightness value.

### palette.set()

Mixin to add or update colors in the palette. Accepts a single key/value pair or a map of multiple color seeds. After setting or updating colors, the palette is automatically rebuilt and all related custom property values are updated.

<CodeExample lang="scss">
  ```scss
  // Set a single color seed
  @include palette.set("primary", hsl(152deg 60% 50%));

  // Set multiple color seeds at once
  @include palette.set((
    "secondary": hsl(214deg 50% 50%),
    "neutral": hsl(214deg 20% 50%)
  ));
  ```
</CodeExample>

### palette.get()

Function to retrieve a color from the palette. You can get a color by name, specify a lightness value, and include an alpha channel.

<CodeExample lang="scss">
  ```scss
  // Get the default value for a color
  @debug palette.get("primary");
  // Debug: var(--vb-primary-50)

  // Get a color at a specific lightness
  @debug palette.get("primary", 80);
  // Debug: var(--vb-primary-80)

  // Get a color at a specific lightness and alpha channel
  @debug palette.get("primary", 80, 20%);
  // Debug: hsl(var(--vb-primary-hs) 80 / 20%)
  ```
</CodeExample>

It's also possible to return the seeds map or a specific seed value directly.

<CodeExample lang="scss">
  ```scss
  // Get the entire seeds map
  @debug palette.get("seeds");
  // Debug: ("primary": hsl(152, 60%, 50%), "secondary": hsl(214, 50%, 50%), "neutral": hsl(214, 20%, 50%), "important": hsl(0, 80%, 50%))

  // Get the original seed value
  @debug palette.get("primary", "seed");
  // Debug: hsl(152, 60%, 50%)
  ```
</CodeExample>

### palette.remove()

Mixin to remove one or more colors from the palette. Accepts a single name or a list of names. After removing colors, the palette is automatically rebuilt to reflect the changes.

<CodeExample lang="scss">
  ```scss
  // Remove a single color
  @include palette.remove("primary");

  // Remove multiple colors
  @include palette.remove("secondary", "neutral");
  ```
</CodeExample>
