Themes are how and where the design system's palette are applied. It also manages a projects light and dark mode color values.

<CodeExample lang="scss">
  ```scss
  @use "@vrembem/core/palette";
  ```
</CodeExample>

## Basic usage

- `theme.set()`
- `theme.remove()`
- `theme.output-theme()`
- `theme.output()`
