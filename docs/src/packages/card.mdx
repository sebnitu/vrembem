## Basic usage

Cards are a highly composable container component that comes with a variety of elements. The three most basic are:

- `card__body`
- `card__image`
- `card__title`

<CodeExample>
  <div slot="output">
    <div class="card">
      <img class="card__image" src="https://picsum.photos/id/335/600/300" />
      <div class="card__body">
        <h2 class="card__title">Card Title</h2>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
    </div>
  </div>
  ```html
  <div class="card">
    <img class="card__image" src="..." />
    <div class="card__body">
      <h2 class="card__title">...</h2>
      ...
    </div>
  </div>
  ```
</CodeExample>

For creating more distinct sections in a card, header and footer elements can be used. It's also possible to have multiple `card__body` elements in your cards as needed.

- `card__header`
- `card__footer`

<CodeExample>
  <div slot="output" class="spacing">
    <div class="card">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body spacing">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
        <p class="foreground-lighter flex justify-content-between">
          <span>August 20th, 2010</span>
          <span>30 comments</span>
        </p>
      </div>
      <div class="card__footer flex justify-content-between">
        <button class="button button_color_primary">Read More</button>
        <button class="button button_color_secondary button_icon">
          <Icon name="heart" />
        </button>
      </div>
    </div>
  </div>
  ```html
  <div class="card">
    <div class="card__header">
      <h2 class="card__title">...</h2>
    </div>
    <div class="card__body">
      ...
    </div>
    <div class="card__footer">
      ...
    </div>
  </div>
  ```
</CodeExample>

### Backgrounds

Card background and screen elements are displayed behind other card elements. These can be paired with the `vb-theme-light` or `vb-theme-dark` theme classes to create a light or dark card theme.

- `card__background`
- `card__screen`

<CodeExample>
  <div slot="output" class="spacing">
    <div class="card vb-theme-light">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
      <img src="https://picsum.photos/id/976/600/300" class="card__background" />
      <div class="card__screen"></div>
    </div>
    <div class="card vb-theme-dark">
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci, id venenatis nisi quam non turpis.</p>
      </div>
      <img src="https://picsum.photos/id/976/600/300" class="card__background" />
      <div class="card__screen"></div>
    </div>
  </div>
  ```html
  <div class="card">
    ...
    <img src="..." class="card__background" />
    <div class="card__screen"></div>
  </div>
  ```
</CodeExample>

### Anchor cards

When creating a card using the anchor `<a>` element, additional styles are applied to make the card more visually interactive with hover and focus styles.

<CodeExample>
  <div slot="output" class="spacing">
    <a href="#" class="card">
      <img class="card__image" src="https://picsum.photos/id/106/600/300" />
      <div class="card__header">
        <h2 class="card__title">Card Title</h2>
      </div>
      <div class="card__body spacing">
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus hendrerit, lacus vel consectetur placerat, nisl eros malesuada orci id venenatis nisi quam non turpis... <strong>Continue Reading</strong></p>
      </div>
    </a>
  </div>
  ```html
  <a href="#" class="card">
    ...
  </a>
  ```
</CodeExample>
