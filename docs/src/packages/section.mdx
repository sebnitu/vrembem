## Basic usage

Sections are composed using the base wrapper (`section`) along with three optional elements. The `section__container` element's primary purpose is to wrap content with a max-width and center using margin auto. The other two element's—`section__background` and `section__screen`—add a background image and colored screen that fills the full size of the section component.

- `section`
  - `section__container`
  - `section__background`
  - `section__screen`

<CodeExample flush>
  <Fragment slot="output">
    <section class="section">
      <div class="section__container spacing text-align-center">
        <h1 class="h1">Section example</h1>
        <p>This is a demonstration of the section component using a screen and background image. The screen, background and foreground colors can be adjusted using CSS variables.</p>
        <hr class="sep margin-y-xl" />
        <div class="flex justify-content-center">
          <button class="button button_color_primary">First action</button>
          <button class="button button_color_secondary">Second action</button>
        </div>
      </div>
      <img class="section__background" src="https://picsum.photos/id/335/600/300" />
      <div class="section__screen"></div>
    </section>
  </Fragment>
  ```html
  <section class="section">
    <div class="section__container">
      ...
    </div>
    <img class="section__background" src="..." />
    <div class="section__screen"></div>
  </section>
  ```
</CodeExample>

Section screens and backgrounds are displayed behind the section container. These can be paired with the `vb-theme-light` or `vb-theme-dark` theme classes to create a light or dark section theme.

## section_padding_[key]

Sections have a few size modifier options to help adjust the space that is used based on how prominent the section needs to be. These are optimized for all screen sizes to avoid oversized areas on mobile.

<CodeExample flush>
  <Fragment slot="output">
    <section class="section section_padding_xl">
      <div class="section__container spacing text-align-center">
        <h1 class="h1">Section example</h1>
        <p>This is a demonstration of the section component using a screen and background image. The screen, background and foreground colors can be adjusted using CSS variables.</p>
        <hr class="sep margin-y-xl" />
        <div class="flex justify-content-center">
          <button class="button button_color_primary">First action</button>
          <button class="button button_color_secondary">Second action</button>
        </div>
      </div>
      <img class="section__background" src="https://picsum.photos/id/335/600/300" />
      <div class="section__screen"></div>
    </section>
  </Fragment>
  ```html
  <section class="section section_padding_xl">
    ...
  </section>
  ```
</CodeExample>

**Available variants**

* `section_padding_sm`
* `section_padding_md`
* `section_padding_lg`
* `section_padding_xl`
